# invoice-generator

Create and download professional invoices as PDF.

## Features

- Your business info, logo, and branding
- Bill-to details
- Line items: description, quantity, rate
- Tax rate and discount
- Payment terms and notes
- Download as PDF (client-side, no server)

## Run

```bash
npm install && npm run dev
```

PDF generation uses jsPDF. No data leaves the browser.
