# Invoice Generator

Create professional invoices with itemized billing, tax calculation, and one-click print or PDF export.

## Features

- Business and client info fields
- Itemized line items with quantity, unit price, and description
- Tax rate and discount fields
- Subtotal, tax, and total auto-calculation
- Invoice number and due date
- Print-ready layout and PDF export
- Save drafts to localStorage
